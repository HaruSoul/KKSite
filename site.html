Privacy Policy for KioskKast

Last Updated: December 2024

Introduction

This Privacy Policy describes how KioskKast ("we", "our", or "the app") collects, uses, and protects information when you use our digital signage and screen sharing application. We are committed to protecting your privacy and ensuring the security of any information processed through our app.

Information We Collect

1. Device Information

  • Device identifiers: We collect device model, operating system version, and unique device identifiers solely for app functionality
  • Network information: Local IP addresses and network configuration for device discovery and communication
  • Storage information: File system paths for media management (photos, videos)

2. Media Content

  • Local media files: Access to photos and videos stored on your device for slideshow functionality
  • Shared media: Content from network shares (SMB) that you explicitly configure
  • Screen content: Temporary access to screen content during active screen sharing sessions

3. Network Communication Data

  • Local network discovery: IP addresses and device names on your local network for tablet discovery
  • WebRTC connection data: Temporary session data for real-time screen sharing
  • Configuration settings: User preferences and connection settings stored locally on your device

How We Use Information

We use the collected information exclusively for:

1. Core Functionality

  • Discovering and connecting to other KioskKast devices on your network
  • Displaying media content in slideshow mode
  • Enabling screen sharing between devices
  • Synchronizing media files from network shares

2. App Performance

  • Maintaining connection quality during screen sharing
  • Optimizing media playback and transitions
  • Managing local cache for network media

3. User Experience

  • Saving your preferences and settings
  • Remembering previously connected devices
  • Auto-connecting to preferred devices

Data Storage and Security

Local Storage Only

  • All data is stored locally on your device
  • No data is transmitted to external servers
  • Settings and preferences are stored in app-specific directories

Network Communication

  • All communication occurs only within your local network
  • WebRTC connections are peer-to-peer between your devices
  • No data passes through third-party servers

Security Measures

  • Network communications use standard encryption protocols
  • Media files are accessed with user-granted permissions only
  • No authentication credentials are stored in plain text

Data Sharing

We do not share any data with third parties. Specifically:

  • No analytics or tracking services are used
  • No advertising networks are integrated
  • No user data is sold, rented, or disclosed to external parties
  • All functionality operates entirely within your local network

Permissions Usage

Required Permissions and Their Usage:

  1. Storage/Media Access
    • Purpose: Read photos and videos for slideshow display
    • Usage: Only accessed when you select media for presentation
  2. Network Access
    • Purpose: Discover devices and enable communication
    • Usage: Local network scanning and peer-to-peer connections
  3. Camera/Microphone (if applicable)
    • Purpose: Screen recording for presentation mode
    • Usage: Only active during user-initiated screen sharing
  4. Wake Lock
    • Purpose: Keep screen active during presentations
    • Usage: Prevents device sleep during active slideshow or streaming
  5. Internet
    • Purpose: WebRTC connections for screen sharing
    • Usage: Local network connections only, no internet servers

Children's Privacy

KioskKast is a general-purpose tool and does not knowingly collect or target information from children under 13. The app does not collect personal information that would require parental consent.

Data Retention

  • Configuration data: Retained until app uninstallation
  • Cached media: Managed by user-configurable cache settings
  • Connection history: Kept for 14 days (configurable) for convenience
  • Temporary session data: Cleared when connections end

User Rights and Control

You have complete control over your data:

  1. Access: All settings and data are accessible within the app
  2. Modification: Change any settings or preferences at any time
  3. Deletion: Clear cache, history, or uninstall to remove all data
  4. Portability: Configuration can be reset to defaults at any time

Changes to This Policy

We may update this Privacy Policy to reflect changes in our app or legal requirements. We will notify users of significant changes through app updates. Continued use of KioskKast after changes constitutes acceptance of the updated policy.
